# NginxInstallTutorial


## 1. 安裝 Homebrew 
* 打開終端機
* 輸入 ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
* 第一次停止的時候 按下RETURN鍵繼續安裝，按下其他間則為終止


## 2. 安裝Nginx
* 輸入 brew install nginx
